# Flag Pie

A small daily country-guessing game. The chart shows flag-color proportions, then players get four escalating facts after wrong guesses.

## Run locally

Open `index.html` in a browser, or from this folder run:

```sh
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Publish free with GitHub Pages

1. Create a new GitHub repository named `flag-pie` and upload these three files.
2. In the repository, open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**, select `main` and `/ (root)`, then save.
4. Within a minute or two GitHub will show the live address, usually `https://YOUR-USERNAME.github.io/flag-pie/`. Copy that link and text it to friends.

The game is static: no accounts, database, hosting bill, or API keys are required.
